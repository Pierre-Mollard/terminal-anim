# Intro

This project is inspired by [Terminal Text Effects](https://github.com/ChrisBuilds/terminaltexteffects).

## Help

The build system for this project is CMake.
Setup build folders:
'''bash
cmake -B build
'''

Build project (same as 'make' when using Makefiles):
'''bash
cmake --build build
'''
